1. ¿El formulario tiene un título claro?
Respuesta: Si, el formulario cuenta con un titulo visible que indica claramente su proposito

2. ¿El formulario explica para qué se usará la información?
Respuesta: Si. se incluye una breve explicación sobre el uso de la información proporcionada por el usuario

3. ¿Cada campo tiene un label o etiqueta visible?
Respuesta: Si, todos los campos tienen etiquetas visibles que facilitan su compresion

4. ¿El lenguaje usado es sencillo y comprensible?
Respuesta: Si, se utiliza un lenguaje claro , sencillo y facil de entender para cualquier usuario

5. ¿El formulario evita pedir datos innecesarios?
Respuesta: Si, solo se solicitan datos relevantes para el objetivo del formulario 

6. ¿El campo principal permite escribir una propuesta o mensaje de forma clara?
Respuesta: Si, Se incluyen un campo amplio que permite al usuario expresar su propuesta o mensaje sin limitaciones

7. ¿El formulario incluye una casilla o mensaje de consentimiento?
Respuesta: Si, se incluye una casilla donde el usuario acepta el uso de sus datos

8. ¿El botón de envío tiene un texto claro?
Respuesta: Si, el boton tiene un texto claro "enviar"

9. ¿La página se puede leer con facilidad?
Respuesta: Si, el diseño tiene buen contraste, tamaño de letra adecuado y es facil de leer

10. ¿Qué mejorarías del formulario para que más personas puedan usarlo?
Respuesta: Se podria mejorar agregando soporte para lecciones de pantalla, mejorando el contarste de colores y optimizando la visualizacion en dispositivos moviles
